## How to Start

1. **Install Java**: Download and install the [Java JDK](https://adoptopenjdk.net/) (version 8 or above).
2. **Install IntelliJ IDEA**: Download [IntelliJ IDEA](https://www.jetbrains.com/idea/download/) and open this project.
3. **Open Main.java**: Go to `src/Main.java`.
4. **Choose a topic**: Uncomment the line for the topic you want to run (e.g. `Variable.Variables.main();`).
5. **Run the program**: Right-click on `Main.java` and select 'Run', or use the green play button.

## Source

In this repo we have a ton of helpful classes for Java beginners, that explain certain things in Java.

Everything should be in the **`src`** folder. The rest of the files are **IntelliJ IDEA** project files

**CLASSES**

 **Main `Main.java`**
> Main class, where all methods are initialized. Now includes a menu and Hello World example.

**Variables `Variable\Variables.java`**
> Variable.Variables such as **`a, b, c, 1, 2, 3, etc.`** Stores the value of integers and Strings for example.

**Swap Variable `Variable\Swapvariable.java`**
> The art of swapping variable information using a temporary Variable **`(swapping x value to y using z)`**

**Strings `Variable\StringStuff.java`**
> A lot of information about Strings and what you can do with them. **`equals` `equalsIgnoreCase` `length` `charAt` `indexOf` `isEmpty` `toUpperCase` `toLowerCase` `trim` `replace`**

**Math `Math\Expressions.java`**
> Math expressions such as **`+, -, *, /, %, ++, --, etc.`**

**GUI.GUI `GUI.GUI.java`**
> Simple use of the Swing graphical package for getting, storing and showing user input and messages in a graphical interface.

**Math `Math\MathStuff.java`**
> Basic math introduction & calculating the hypotenuse of a right side triangle. Also using the **`Swing GUI.GUI`** to get user input.

**Random `Math\RandomStuff.java`**
> Pseudo-random number generator between 1-5.

**Switches `Loops\Switches.java`**
> Using switches to compare for a value, instead of having to have a large tree of if-statements

**Input `Variable\Input.java`**
> The use of getting and storing user input via a Scanner.

**Logical Operators `Variable\LogicalOperators.java`**
> Using logical operators like **`&&, ||, !`** to check if-statements and case statements. Also using methods to infinitely run the program until the correct key or phrase is input.

**Loops `Loops\WhileLoops.java`, `Loops\ForLoops.java`, `Loops\ForEachLoops.java`, `Loops\NestedLoops.java`**
> Examples of different loop types and how to use them.

**Arrays `Array\Arrays.java`, `Array\Array2D.java`**
> How to store and work with multiple values in arrays, including 2D arrays.

**ArrayLists `ArrayList\ArrayLists.java`, `ArrayList\ArrayList2D.java`**
> Dynamic lists and 2D ArrayLists for flexible data storage.

**Methods `Methods\Methods.java`, `Methods\OverloadedMethods.java`**
> Creating reusable code blocks and overloaded methods.

**Wrapper Classes `WrapperClass.java`**
> Using primitive types as objects.

## Källa

I detta repo finns många hjälpsamma klasser för Java-nybörjare, som förklarar olika saker i Java.

Allt finns i **`src`**-mappen. Resten av filerna är **IntelliJ IDEA** projektfiler.

**KLASSER**

 **Main `Main.java`**
> Huvudklass där alla metoder initieras. Nu med meny och Hello World-exempel.

**Variabler `Variable\Variables.java`**
> Variabler som **`a, b, c, 1, 2, 3, etc.`**. Lagrar värden för t.ex. heltal och strängar.

**Byta Variabel `Variable\Swapvariable.java`**
> Konsten att byta variabelinformation med en temporär variabel **`(byta x värde till y med z)`**

**Strängar `Variable\StringStuff.java`**
> Information om strängar och vad man kan göra med dem. **`equals` `equalsIgnoreCase` `length` `charAt` `indexOf` `isEmpty` `toUpperCase` `toLowerCase` `trim` `replace`**

**Matematik `Math\Expressions.java`**
> Matematiska uttryck som **`+, -, *, /, %, ++, --, etc.`**

**GUI.GUI `GUI.GUI.java`**
> Enkel användning av Swing för att ta emot, lagra och visa användarinmatning och meddelanden i ett grafiskt gränssnitt.

**Matematik `Math\MathStuff.java`**
> Grundläggande matematik & beräkning av hypotenusan på en rätvinklig triangel. Använder även **`Swing GUI.GUI`** för inmatning.

**Slumptal `Math\RandomStuff.java`**
> Slumptalsgenerator mellan 1-5.

**Switchar `Loops\Switches.java`**
> Användning av switch för att jämföra värden istället för många if-satser.

**Inmatning `Variable\Input.java`**
> Hur man tar emot och lagrar användarinmatning via Scanner.

**Logiska Operatorer `Variable\LogicalOperators.java`**
> Användning av logiska operatorer som **`&&, ||, !`** för att kontrollera if-satser och case-satser. Använder även metoder för att köra programmet tills rätt nyckel eller fras matas in.

**Loopar `Loops\WhileLoops.java`, `Loops\ForLoops.java`, `Loops\ForEachLoops.java`, `Loops\NestedLoops.java`**
> Exempel på olika looptyper och hur de används.

**Arrayer `Array\Arrays.java`, `Array\Array2D.java`**
> Hur man lagrar och arbetar med flera värden i arrayer, inklusive 2D-arrayer.

**ArrayLists `ArrayList\ArrayLists.java`, `ArrayList\ArrayList2D.java`**
> Dynamiska listor och 2D ArrayLists för flexibel datalagring.

**Metoder `Methods\Methods.java`, `Methods\OverloadedMethods.java`**
> Skapa återanvändbara kodblock och överlagrade metoder.

**Wrapper-klasser `WrapperClass.java`**
> Användning av primitiva typer som objekt.
